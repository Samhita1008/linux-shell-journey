# Linux Shell Journey

A running log of shell scripting practice — daily exercises in Bash covering file management, process monitoring, permissions/security auditing, cron automation, and system health checks.

This is a learning repo, not a polished project: scripts are small, single-purpose, and written while working through specific Linux/Bash concepts rather than as one cohesive application.

## Structure

- **`revision/`** — organized day-by-day exercises (file manager, task manager, process monitor, system health)
- **Root-level `.sh` files** — standalone practice scripts grouped loosely by topic: file utilities, permission/security checks, cron jobs, system monitoring, and small CLI tools (calculator, password strength checker, voting system, etc.)

## Topics covered

- File and directory operations
- Process monitoring and management
- Permission auditing (SUID/SGID, world-writable files, ownership)
- Cron-based automation (disk logging, health checks, cleanup jobs)
- Basic system info and resource monitoring
- Small CLI utilities for practicing argument handling, input validation, and logging

## Note

These scripts were written as individual practice exercises over time, not reviewed as a single system — expect variation in style and some overlap between similarly-named scripts as concepts were revisited.
